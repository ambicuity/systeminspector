Overview
Node.js system information package. Lightweight collection of 58 core inspection functions to retrieve detailed hardware, system and OS information. For Linux, macOS, partial Windows, FreeBSD, OpenBSD, NetBSD, SunOS and Android support. Dependency free. Works with Node.js 18+, Bun and Deno
systeminspector 1.0.0 is the neutral initial release.
